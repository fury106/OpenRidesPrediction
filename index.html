OpenRidesPrediction is a simple C++ desktop app which calculates the probability that certain rides at certain theme parks will open depending the (extreme) weather conditions. At this moment, only one park (Plopsaland De Panne) is supported, but in the future more parks will be added to the app. The app is currently only available in dutch, but more languages will be added in the future.

PC specs

In order for the app to work properly, these are the minimum pc specs required.

  • Any 64 bit version of Windows
  • Minimum disk space: 40 MB
